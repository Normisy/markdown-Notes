# C2. 线性表
## 2.1 线性表数据结构
线性表也称有序表，**它的每一个实例都是元素的一个有序集合，形式为$(e_0,e_1,\cdots ,e_{n-1})$，其中n为有穷自然数，$e_i$是线性表的元素，i是元素的==索引==，n是线性表的长度**
**可以认为$e_0$先于$e_1$，$e_1$先于$e_2$，……，该先后关系是线性表的唯一关系**

## 2.2 抽象数据类型linearList
一个线性表可以用一个ADT来说明其实例和操作，抽象数据类型的说明保证了程序语言描述的有效性，所有满足抽象数据类型说明的语言描述，都可以在应用中替换使用

### 2.2.1 抽象类linearList
抽象类包含着没有实现代码的成员函数，这就是**纯虚函数**，用`......=0`作为初始值来说明。
我们可以建立抽象类的对象指针，但能对具体类建立实例或对象
下面是抽象类linearList的代码：
```
template<class T>
class linearList {
public:
	virtual ~linearList(){} = 0;
	virtual bool empty() const = 0;
	virtual int size()
}
```
